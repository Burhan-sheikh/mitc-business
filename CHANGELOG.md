# Changelog

All notable changes to MITC Web App will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-23

### Added
- Initial release of MITC Web App
- Product browsing with grid and list views
- Product detail pages with image galleries
- Smart search and filtering system
- Live chat using Firebase RTDB
- Reviews and ratings system
- User authentication (Email/Password + Google OAuth)
- Admin dashboard with analytics
- Product management (CRUD operations)
- Image upload with compression and Cloudinary integration
- Chat moderation features
- User management and role-based access control
- CSV/JSON import and export
- 6 premium theme presets
- PWA support with offline functionality
- Responsive design (desktop-first)
- Store information page
- Account deletion workflow
- Firebase security rules
- Cloud Functions for image processing and cleanup
- Comprehensive documentation

### Features
- **For Users**:
  - Browse products with filters
  - View detailed product information
  - Real-time chat support
  - Submit and view reviews
  - Guest and authenticated browsing
  - Theme customization
  - PWA installation

- **For Admins**:
  - Dashboard with analytics
  - Product management
  - Chat moderation
  - User management
  - Review moderation
  - Import/Export tools
  - Settings configuration

### Technical
- React 18 + Vite
- Firebase (Auth, Firestore, RTDB, Functions)
- Tailwind CSS
- Framer Motion
- Cloudinary integration
- PWA with service worker
- Role-based security
